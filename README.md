## Tech Stack

- **Streamlit**: An open-source app framework for building interactive web applications.
- **Pandas**: A data manipulation library for handling and analyzing structured data.
- **NumPy**: For fast numerical computations and array operations.
- **Matplotlib**: For creating customizable static visualizations and plots.
- **Seaborn**: For high-level statistical visualizations built on top of Matplotlib.
- **Scikit-learn**: For implementing machine learning models, preprocessing, and evaluation.


## Getting Started

1. **Clone this repository:**
    ```
    git clone https://github.com/yourusername/pluton.git
    cd pluton
    ```

2. **Install dependencies:**
    ```
    pip install -r requirements.txt
    ```

3. **Run locally:**
    ```
    streamlit run app.py
    ```

4. **Or, visit [pluton.streamlit.app](https://pluton.streamlit.app) to try it instantly (no install needed)!**
